3. Check if the patch is already applied by grepping for the marker line added by the patch. The marker is the comment string `// discord-local-scoping patch applied`. If found, print `discord-channel: patch already applied — skipping` and exit 0.

4. Check if `$PATCH_FILE` exists. If not, print `discord-channel: patch file not found at $PATCH_FILE — skipping` and exit 0. (Graceful: the patch file is written by Plan 01-02; this script runs on every session start so it must tolerate being run before the patch is generated.)

5. Check if `$SERVER_TS` exists. If not, print `discord-channel: server.ts not found at $SERVER_TS — plugin not installed, skipping` and exit 0.

6. Apply the patch:
   ```bash
   cd "$HOME" && patch -p0 < "$PATCH_FILE"
   ```
   The patch headers use absolute-like paths starting with `.claude/plugins/cache/...` so `cd $HOME` makes `-p0` resolve correctly. Print result to stderr.

7. If patch succeeds (exit 0): print `discord-channel: local-scoping patch applied successfully`.
   If patch fails: print `discord-channel: patch failed — check $PATCH_FILE` and exit 1.

Make the script executable: `chmod +x scripts/apply-discord-patch.sh`.

Full script skeleton:
```bash
#!/usr/bin/env bash
# discord-local-scoping apply script
# Idempotent — safe to run multiple times. Fires from Claude Code SessionStart hook.
# Applies patches/discord-local-scoping.patch to the discord plugin cache.

set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PATCH_FILE="$SCRIPT_DIR/../patches/discord-local-scoping.patch"
SERVER_TS="$HOME/.claude/plugins/cache/claude-plugins-official/discord/0.0.1/server.ts"
MARKER="// discord-local-scoping patch applied"

# Already applied?
if grep -qF "$MARKER" "$SERVER_TS" 2>/dev/null; then
  echo "discord-channel: patch already applied — skipping" >&2
  exit 0
fi

# Patch file present?
if [[ ! -f "$PATCH_FILE" ]]; then
  echo "discord-channel: patch file not found at $PATCH_FILE — skipping" >&2
  exit 0
fi

# Plugin installed?
if [[ ! -f "$SERVER_TS" ]]; then
  echo "discord-channel: server.ts not found — plugin not installed, skipping" >&2
  exit 0
fi

cd "$HOME"
if patch -p0 < "$PATCH_FILE"; then
  echo "discord-channel: local-scoping patch applied successfully" >&2
else
  echo "discord-channel: patch failed — check $PATCH_FILE" >&2
  exit 1
fi
```
