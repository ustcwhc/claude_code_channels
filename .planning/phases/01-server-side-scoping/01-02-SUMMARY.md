- .mcp.json now injects `DISCORD_PROJECT_DIR: ${CLAUDE_PROJECT_DIR}` so Claude Code passes the project directory to the MCP server

- `readAccessFile()` and `saveAccess()` both use `ACTIVE_ACCESS_FILE` — all reads and writes go to whichever file was resolved at boot
